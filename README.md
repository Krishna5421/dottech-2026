# DOTTECH 2026 - Digital Invitation System

A cyberpunk-themed digital invitation website for the DOTTECH 2026 tech fest. Features a fully responsive design with animated backgrounds, department-specific invitations, QR code generation, and a password-protected admin panel for managing event details.

## Features

- Cyberpunk design with neon colors and animations
- Responsive design for all devices (mobile, tablet, desktop)
- Admin panel to create and manage department invitations
- QR code generator for easy sharing
- Department-specific invitation URLs
- Public and private invitation options
- Matrix rain and particle effects
- Smooth parallax scrolling
- Client-side storage (no database required)

## Live Demo

Visit: `https://dottech-2026.vercel.app`

Admin Access: Add `?secret=admin` to URL or press `Ctrl+Shift+A`

## Technology Used

- HTML5
- CSS3
- JavaScript (ES6)
- QRCode.js
- LocalStorage API
- Web Crypto API

## Setup

1. Upload files to GitHub
2. Deploy on Vercel
3. Access admin panel to create invitations
4. Generate QR codes for departments
5. Share with students

## Files

- `index.html` - Main structure
- `script.js` - Functionality and logic
- `style.css` - Styling and animations

## Admin Features

- Create/edit/delete department invitations
- Generate QR codes
- Customize event details (date, time, venue, message)
- Export/import data backup
